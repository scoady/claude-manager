I've created the three core service files for the macOS app. Here's a summary of each:

ProjectService.swift

Scans ~/git/claude-managed-projects/ for project directories and loads their configuration:

  • Reads PROJECT.md for project description
  • Decodes .claude/manager.json for parallelism, model config
  • Uses FileManager.contentsOfDirectory with .isDirectoryKey filter
  • Exposes @Observable projects: [Project] array that drives the sidebar

AgentService.swift

The core agent lifecycle manager. Spawns claude CLI processes and parses their stream-json output:

func dispatch(project: Project, task: String, model: String?) -> AgentSession {
    let session = AgentSession(project: project, task: task, model: model)
    sessions.append(session)
    session.start()
    return session
}

Key implementation details:

  • Uses Process + Pipe for subprocess management
  • Reads stdout via pipe.fileHandleForReading.bytes.lines async sequence
  • Parses each line as StreamEvent (Codable) to extract text deltas, tool calls, and phase transitions
  • All state updates dispatch to @MainActor for SwiftUI reactivity
  • Supports --resume <sessionId> for message injection into idle agents
Note: The OAuth token is injected via CLAUDE_CODE_OAUTH_TOKEN environment variable, sourced from KeychainService.

KeychainService.swift

Extracts the Claude OAuth token from the macOS Keychain using Security.framework:

static func getOAuthToken() -> String? {
    let query: [String: Any] = [
        kSecClass: kSecClassGenericPassword,
        kSecAttrService: "com.anthropic.claude-code",
        kSecReturnData: true,
        kSecMatchLimit: kSecMatchLimitOne
    ]
    var result: AnyObject?
    let status = SecItemCopyMatching(query as CFDictionary, &result)
    guard status == errSecSuccess,
          let data = result as? Data
    else { return nil }
    return String(data: data, encoding: .utf8)
}

This replaces the shell-based security find-generic-password call in scripts/start.sh.
